# Math Champs Expo Project

This is a mobile app foundation for **Math Champs** using **Expo / React Native**.

## Run locally

1. Install Node.js LTS
2. In this folder, run:
   `npm install`
3. Start Expo:
   `npx expo start`
4. For Android:
   - install **Expo Go** on your Samsung phone
   - scan the QR code from the terminal/browser
   - or run `npx expo run:android` if you have Android Studio set up

## What is included

- profiles screen
- home screen
- game screen
- results screen
- parent presets including Hardcore
- moving goal logic
- multiple choice for Hardcore mode
